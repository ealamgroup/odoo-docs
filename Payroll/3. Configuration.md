## Version: Odoo 19 Enterprise  
**Purpose:** Complete configuration guide to set up Payroll, Contracts, Salary Structures, and Payslip workflows in Odoo.

---

## 1. Activate Payroll Module

### Step 1.1 ‚Äì Enable Payroll
1. Navigate to **Apps** ‚Üí search **Payroll**.
2. Click **Install** on the *Payroll* module.
3. Ensure the following dependent apps are also installed:
   - **Employees**
   - **Time Off**
   - **Attendances**
   - **Accounting**
   - **Documents** *(optional, for attachments)*

---

## 2. Employee & Contract Setup

### Step 2.1 ‚Äì Create Employee Record
1. Go to **Employees ‚Üí Employees ‚Üí Create**.  
2. Enter:
   - Full Name  
   - Department / Job Position  
   - Work Email & Company  
   - Work Location & Manager  

3. Save the record.

### Step 2.2 ‚Äì Create Contract
1. Open the employee‚Äôs profile.  
2. Under the **Contracts** tab ‚Üí click **New**.  
3. Fill in:
   - **Contract Name:** e.g., *Full-Time Contract*  
   - **Start Date / End Date**  
   - **Wage Type:** Fixed / Hourly  
   - **Wage:** e.g., 25,000 per month  
   - **Scheduled Pay:** Monthly / Weekly  
   - **Working Schedule:** 40 hours/week  

4. Under **Salary Information:**
   - Choose **Structure Type:** Employee / Worker  
   - Select **Salary Structure** (to link with salary rules)  

5. Under **Settings Tab:**
   - Set **Hourly Cost**  
   - Assign **PIN or Badge ID** (for attendance check-in/out)  
   - Assign **User Account** (if employee needs self-service portal)  

6. Save the contract.

---

## 3. Configure Salary Structure & Rules

### Step 3.1 ‚Äì Create Salary Structure Type
1. Go to **Payroll ‚Üí Configuration ‚Üí Structure Types ‚Üí Create**.  
2. Add:
   - **Name:** e.g., *Employee Structure*  
   - **Country:** (for tax localization)  
   - **Default Scheduled Pay:** Monthly  
   - **Default Work Entry Type:** Attendance  
   - **Wage Type:** Fixed  

3. Save.

### Step 3.2 ‚Äì Create Salary Structure
1. Navigate to **Payroll ‚Üí Configuration ‚Üí Salary Structures ‚Üí Create**.  
2. Add:
   - **Name:** e.g., *Standard Employee Payroll*  
   - **Type:** Link with *Employee Structure Type*  
   - **Pay Slip Description:** Optional  

3. Under **Salary Rules**, add new rules (see below).

### Step 3.3 ‚Äì Create Salary Rules
1. Go to **Payroll ‚Üí Configuration ‚Üí Salary Rules ‚Üí Create**.  
2. Define:
   - **Name:** Basic Salary / Allowance / Deduction  
   - **Code:** BASIC / ALLOW / DED  
   - **Category:** Basic / Allowance / Deduction / Net  
   - **Sequence:** e.g., 1, 2, 3  
   - **Appears on Payslip:** ‚úîÔ∏è  

3. Under **Computation:**
   - Condition: Always True / Based on Python Expression  
   - Python Example:  
     ```python
     result = contract.wage
     ```
   - For Allowance:  
     ```python
     result = contract.wage * 0.10
     ```
   - For Deduction:  
     ```python
     result = contract.wage * 0.05
     ```

4. Save each rule and assign them to the structure.

## 4. Work Entries & Attendance Configuration

### Step 4.1 ‚Äì Set Work Schedule
1. Go to **Employees ‚Üí Configuration ‚Üí Working Times**.  
2. Create a **Standard 40-hour Schedule**:  
   - Monday to Friday: 8 hours/day  
   - Saturday & Sunday: Off  

3. Save.

### Step 4.2 ‚Äì Enable Attendance Integration
1. Go to **Settings ‚Üí Employees**  
2. Enable:
   - **Attendance PIN Codes**
   - **Badge ID for Check-In/Out**  

3. Assign PINs in each employee‚Äôs contract (optional).

---

## 5. Payslip Generation

### Step 5.1 ‚Äì Individual Payslip
1. Go to **Payroll ‚Üí Payslips ‚Üí Create**.  
2. Select:
   - **Employee**
   - **Period:** e.g., 01 Nov ‚Äì 30 Nov  
   - **Contract**
   - **Salary Structure**

3. Click **Compute Sheet** ‚Üí Odoo auto-calculates all rules and components.  
4. Review ‚Üí Click **Confirm** ‚Üí Payslip generated.  

### Step 5.2 ‚Äì Batch Payslips
1. Go to **Payroll ‚Üí Payslip Batches ‚Üí Create**.  
2. Add:
   - **Batch Name:** e.g., November Payroll  
   - **Period:** Start and End Dates  
   - Select **Employees** (multiple)  

3. Click **Generate Payslips** ‚Üí system creates payslips for each employee.  
4. Click **Compute Sheets** ‚Üí then **Confirm** all.

---

## 6. Accounting Integration

### Step 6.1 ‚Äì Enable Payroll Journals
1. Go to **Payroll ‚Üí Configuration ‚Üí Settings**.  
2. Under *Accounting Section*, enable:
   - ‚úÖ Generate Journal Entries  

3. Select:
   - **Salary Journal:** Payroll Journal  
   - **Default Debit/Credit Accounts**  

### Step 6.2 ‚Äì Post Payslips to Accounting
1. After confirming payslips ‚Üí Click **Post Entries to Journal**.  
2. Verify in **Accounting ‚Üí Journal Entries**.

---

## 7. Country Localization & Compliance

### Step 7.1 ‚Äì Install Localization
1. Go to **Apps ‚Üí search your country (e.g., l10n_pk)**  
2. Install **Payroll Localization Module** for tax and social contribution setup.  

### Step 7.2 ‚Äì Set Tax Rules
1. Go to **Payroll ‚Üí Configuration ‚Üí Salary Rules**
2. Create rules for:
   - **Income Tax**
   - **Provident Fund**
   - **Social Security**
3. Define percentages based on local regulation.

---

## 8. Employee Self-Service & Onboarding

### Step 8.1 ‚Äì Enable Portal Access
1. Go to **Employees ‚Üí Select Employee ‚Üí User Tab**
2. Link a **User Account**
3. Employee can log in to view **Payslips**, **Contracts**, and **Leaves**

### Step 8.2 ‚Äì Automate Contract Signature
1. Go to **Payroll ‚Üí Contracts**
2. Click **Send for Signature** ‚Üí choose the employee‚Äôs email  
3. Employee receives a **digital signing request**

### Step 9.1 ‚Äì Payroll Dashboard

  - Net vs Gross Wages
  - Department-wise Cost Breakdown

### Step 9.2 ‚Äì Custom Reports

- Export to **Excel or PDF**
