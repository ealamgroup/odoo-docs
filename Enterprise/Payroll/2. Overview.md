## 2. Contract Configuration

### 2.1 Payroll Tab
Defines the **contract details**, including:
- **Contract Duration:** Start and End dates  
- **Wage Type:** Fixed or variable  
- **Scheduled Pay:** e.g., Monthly, Bi-Weekly, etc.  
- **Wage Amount:** e.g., 25,000 per month  
- **Employee Type:** Intern, Full-time, Part-time  
- **Contract Type:** Permanent / Temporary  

### 2.2 Cost & Work Schedule Tracking
- Tracks **Employer Cost** and **Monthly Cost**
- Uses **Working Schedule** (e.g., 40 hours/week) as a work entry source
- Supports detailed **cost breakdown per employee**

### 2.3 New Features (Odoo 19)
Odoo 19 introduces advanced payroll tracking with:
- **Retirement Plan Fields** (401k, Pension, etc.)
- **Benefit Tracking:** Commuter purse slips and reimbursements
- **W4 Federal & State Filing Details:** Tax status and withholding configuration

## 4. Salary Structure and Rules

### 4.1 Salary Structure Types
Salary Structure Types define the **framework** for salary computation, including:
- **Employee Category:** e.g., Regular Employee, Contractor  
- **Fixed Wage & Pay Schedule:** Monthly / Weekly  
- **Working Hours:** Standard 40 hours per week  
- **Work Entry Type:** e.g., Attendance, Leave, Overtime  

### 4.2 Salary Rules
Salary Rules determine **how salary components are calculated**:
- **Examples:** Basic Salary, Allowances, Deductions, Bonus  
- **Rule Parameters:**  
  - **Code** (unique identifier)  
  - **Category** (Basic / Allowance / Deduction)  
  - **Sequence** (calculation order)  
  - **Structure Link** (associated salary structure)  
- Option to display rule values on:
  - **Payslip**
  - **Cost Dashboard**

### 4.3 Computation Methods
- **Condition Types:**
  - Always True
  - Range Based
  - Other Input Dependent
  - Python Expression  
- **Python Scope:**  
  Used for advanced formulas and logic-driven salary computations.  
  Automatically derives payable amounts based on wage and attendance.
